
# Smart Equipment Monitoring Dashboard

## Project Overview
This project analyzes industrial equipment performance using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Dataset Features
- Equipment Name
- Operating Hours
- Downtime Hours
- Failure Type
- Production Output
- Temperature
- Vibration

## Key Insights
- Pump A recorded the highest downtime.
- Turbine A showed the highest average temperature.
- Generator A achieved the highest production output.
- Leak and Electrical failures were among the most common failure types.
